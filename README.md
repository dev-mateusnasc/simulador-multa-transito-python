# 🚓 Simulador de Multa de Trânsito

Projeto desenvolvido em Python para simular a aplicação de multas por excesso de velocidade.

## Funcionalidades

- Verificação de limite de velocidade
- Cálculo de excesso de velocidade
- Classificação da multa
- Cálculo do valor da multa

## Tecnologias

- Python

## Como executar

python simulador_multa.py
